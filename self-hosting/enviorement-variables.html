About

We provide some Environment Variables for docker containers to provide a way to configure them and provide additional, optional configuration which maybe required for self hosting

Database API

Db1Settings – REQUIRED – Settings for database 1 (for the native database driver this means a redis connection string)

Db2Settings –  REQUIRED – Settings for database 2 (for the native database driver this means a postgres connection string)

JwtSecret –  REQUIRED – This sets the Secret of the JWT Bearer Token Generation

SendGridToken – REQUIRED – This sets the SendGrid token for sending out emails (for things like registration)

DbDriverGuid –  OPTIONAL – This sets the GUID for the default database driver (default is 283f10c3-5022-4cda-a5a7-e491a1e84b32)

JwtTimeout – OPTIONAL – This sets the Timeout for the JWT bearer Token Generation (default is 10 minutes)

JwtLLTimeout – OPTIONAL – This sets the Timeout for the Long Lived JWT Bearer token Generation (default is 30 days). We do not recommend setting this to less than 10 days, because some of our code only refreshes the tokens every day to ensure they're still up to date.

EndpointUrl – OPTIONAL – The URL that is put into things like email or return data, we default to https://api.pamaxie.com/db/v1/, if you host the server yourself we recommend setting this to your URL.

Hosts – OPTIONAL – Default port and IP address the application uses (maybe separated by ‘;’ for multiple hosts) (otherwise we default to all IP's on port 80 (http://0.0.0.0))

Scanning API

BaseUrl – OPTIONAL – Sets the Base URL we validate against, when authenticating a users token or checking if the API is available. We default to: https://api.pamaxie.com/
