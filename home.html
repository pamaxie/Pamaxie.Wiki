Contribute to Pamaxie

Before contributing your source code, please make sure that you know and understand our code styles. 
All languages that we end up using with Pamaxies development will have a specific structure we require developers to comply to. 
This is especially important to keep code readable. These guides can be found under the “Code Style / Design Guides” grouping.

Setting up development enviorement

We generally recommend using GitKraken to manage our Git Repositories. Since all projects with Pamaxie are hosted publically accessable, all you need is the free version of it. 
Other software to manage GitHub repositories include GitHub Desktop or the  Git CLI [COMPLEX]. You may also use JetBrains integrated solutions (some people in our team prefer this).

For our Development we use Visual Studio. Some of our Devs also replaced Visual Studio with Jetbrains products (Rider for C#, CLion for C & C++, PyCharm for Python, etc..). 
For Interpreter Languages like Python and JS some of us also use VS Code.

 

Visual Studio Plugins

We have a number of plugins we recommend for Visual Studio Development.

  • ML.NET Model Builder (This is a simple UI tool to build custom machine learning models in VS2019)
  • File Nesting (Automatically nests files based on file name and enables developers to nest / unnest any file manually)
  • VSColorOutput (Color output for build and debug windows)

 

C# Specific Plugins

  • JetBrains ReSharper (Productivity Extension for VS2019)

 

IntelliJ / Jetbrains IDE Plugins

  • DiscordBeam (Share files and code to a Discord Server, we use this for our Internal Discord Server a lot)
  • Heap Allocations Viewer (This highlights local object allocations and more)
  • Nyan Progress Bar (For the flare.)
  • Ideolog (Easily Customizable code highlighting)
  • Rainbow Brackets (Makes functional blocks more easily descernable)
  • Hiberbee Theme (Theme we recommend)
